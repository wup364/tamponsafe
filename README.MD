## 📖 项目简介

TamponSafe 是一款专为女性设计的卫生棉条使用记录与防忘提醒 Web 应用。它采用极简设计，所有数据均存储在本地，确保您的隐私安全。

### ✨ 核心特性

- **🔔 智能提醒** - 实时计时，超时自动发出严重警告
- **🛡️ 防呆设计** - 防止双重棉条风险，智能拦截误操作
- **📊 历史记录** - 自动记录每次使用时间，支持标签和备注
- **💾 数据导出/导入** - 支持备份和恢复数据
- **🔒 隐私优先** - 所有数据本地存储，不上传任何云端
- **📱 响应式设计** - 完美适配手机和桌面浏览器
- **⚙️ 可自定义设置** - 支持自定义最小/最大/警告时长

---

## 🚀 快速开始

### 使用方法

1. **放入棉条**：点击 `+ 刚塞入一根` 按钮，系统自动记录时间
2. **查看状态**：首页实时显示已放置时长和剩余安全时间
3. **取出棉条**：点击 `✕ 已取出` 按钮，记录本次使用数据
4. **查看历史**：展开历史记录查看过往使用记录

### 数据管理

在设置中可以：

- 导出所有数据为 JSON 文件（`🌸 导出数据`）
- 从 JSON 文件导入数据（`🎀 导入数据`）
- 清空所有历史记录

## ⚙️ 设置说明

### 时长设置

| 设置项       | 默认值 | 说明                     |
| ------------ | ------ | ------------------------ |
| 最小建议时长 | 4 小时 | 建议的最短使用时间       |
| 最大建议时长 | 8 小时 | 建议的最长使用时间       |
| 警告阈值     | 8 小时 | 超过此时间将发出严重警告 |

> 💡 提示：支持小数设置，例如 `4.5` 小时 = 4 小时 30 分钟

### 标签系统

记录取出时可以选择标签：

- 🌸 正常
- 🔴 量多
- 🔵 量少
- 😊 舒适
- 💦 漏液
- 😣 腹痛

还可以添加备注（最多 20 字）记录您的感受或状态。

## 🔐 隐私与安全

- **本地存储**：所有数据存储在浏览器的 `localStorage` 中
- **无服务器**：无需联网，无需注册登录
- **数据备份**：支持导出为 JSON 文件，定期备份
- **免责声明**：本工具仅为提醒辅助，不能替代医疗建议。如有不适请立即就医。
